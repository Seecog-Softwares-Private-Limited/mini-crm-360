## Product Overview

**Mini CRM 360** is a powerful, all-in-one Customer Relationship Management (CRM) solution designed to help businesses of all sizes streamline their operations, manage customer relationships, and drive growth. Built with modern web technologies, Mini CRM 360 offers a comprehensive suite of tools for customer management, marketing campaigns, task tracking, invoicing, and business analytics.

### Product Tagline
*"Powerful All-in-One CRM Solution For Your Business Growth"*

### Key Value Propositions
- **Unified Platform**: Manage all your business operations from a single dashboard
- **Cloud-Based**: Access your CRM from anywhere, anytime, on any device
- **Scalable**: Grow from startup to enterprise with flexible pricing plans
- **User-Friendly**: Intuitive interface designed for users of all technical levels
- **Affordable**: Start free and scale as you grow

## Technical Specifications

### Technology Stack

**Backend**:
- **Runtime**: Node.js (v14+)
- **Framework**: Express.js 5.1.0
- **Database**: MySQL 8.0+ with Sequelize ORM
- **Authentication**: JWT (JSON Web Tokens) with refresh tokens
- **File Upload**: Multer for handling file uploads
- **Email**: Nodemailer for transactional emails
- **PDF Generation**: PDFKit
- **Payment Gateway**: Razorpay integration

**Frontend**:
- **Templating Engine**: Handlebars.js
- **CSS Framework**: Bootstrap 5.1.3
- **Icons**: Font Awesome 6.0
- **JavaScript**: Vanilla ES6+
- **Animations**: HTML5 Canvas, CSS3 animations

**Third-Party Integrations**:
- **WhatsApp**: Meta WhatsApp Cloud API
- **Email**: SMTP (configurable)
- **Payment**: Razorpay
- **Social Login**: Google OAuth (optional)

### System Requirements

**Server Requirements**:
- Node.js v14 or higher
- MySQL 8.0 or higher
- Minimum 2GB RAM
- 10GB storage space
- SSL certificate (for production)

**Browser Support**:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

### API Specifications

**Base URL**: `https://yourdomain.com/api/v1`

**Authentication**: Bearer Token (JWT)

**Response Format**: JSON

**Rate Limiting**: Configurable per plan

**API Documentation**: Swagger UI available at `/api-docs`

### Database Schema

**Core Tables**:
- `users` - User accounts and authentication
- `businesses` - Business profiles
- `customers` - Customer contacts
- `campaigns` - Marketing campaigns
- `templates` - Email/WhatsApp templates
- `tasks` - Task management
- `notes` - Notes and timeline entries
- `lead_forms` - Lead capture forms
- `invoices` - Invoice management (premium)
- `plans` - Subscription plans
- `user_plans` - User plan assignments
- `menu_items` - Dynamic menu configuration
- `plan_menu_items` - Plan-specific feature locks

## Getting Started

### Quick Start Guide

1. **Sign Up**
   - Visit the landing page
   - Click "Get Started Free"
   - Complete registration form
   - Verify your email address

2. **Activate Account**
   - Check your email for activation link
   - Click activation link
   - Account automatically activated

3. **Complete Profile**
   - Upload profile picture
   - Add business information
   - Configure settings

4. **Add Customers**
   - Import existing customer data
   - Or manually add customers
   - Organize with tags and categories

5. **Create Campaigns**
   - Choose email or WhatsApp
   - Select template
   - Add recipients
   - Schedule or send immediately

6. **Track Activities**
   - Monitor dashboard
   - Review analytics
   - Manage tasks
   - Track follow-ups

4. Verify data accuracy
5. Start using Mini CRM 360
